Tarefa 1 - Introdução a Programação Para internet e HTML básico

1) O que é a internet ? o que é a web ? são a mesma coisa?
A internet é uma rede mundial que tem como objetivo interligar computadores para fornecer ao usuário o acesso a diversas
informações. Por isso é chamada de rede mundial de computadores.
A internet é uma rede que conecta milhões de computadores pelo mundo e a WEB é uma das várias ferramentas de acesso a rede; WEB = World Wide Web --> WWW;

2) O que é HTTP?
HyperText Transfer Protocol é o significado da sigla HTTP que traduzido para o português é "Protocolo de Transferência de
Hipertexto", HTTP é um protocolo (protocol) que permite a obtenção de recursos, como documentos HTML. É a base de qualquer troca de
dados na Web e um protocolo cliente-servidor, o que significa que as requisições são iniciadas pelo destinatário, geralmente um navegador da Web

3) O que é HTML?
HTML é uma linguagem de marcação --> HyperText Markup Language. É uma linguagem que define os elementos gráficos de uma página web,
usando como uma analogia o corpo humano, o HTML seria o esqueleto. Neste caso, o HTML é o eesqueleto da página WEB;
Linguagens de marcação são notações utilizadas para descrever
aspectos qualquer de documentos por meio de tags;

4) Cite e explique o que são tags HTML.
Tags são estruturas de linguagem de marcação contendo instruções, tendo uma marca de início e outra de fim para que o navegador
possa renderizar uma página.
